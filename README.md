## 📌 Overview

Cyber Network Packet Sniffer is a Python-based cybersecurity project designed to capture, analyze, and monitor live network traffic.

The application captures network packets, extracts important information, identifies communication protocols, and stores packet details for security analysis.

This project was developed as part of my **Cyber Security Internship at Arch Technologies**.


---

## 🎯 Objectives

The main objectives of this project are:

- 🔍 Capture live network packets
- 🌐 Monitor network communication
- 🛡️ Understand TCP/IP traffic
- 📊 Analyze packet information
- 📝 Generate packet logs
- 📚 Learn practical network security concepts

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/Cyber-Network-Packet-Sniffer.git
```


### 2. Navigate to Project Folder

```bash
cd Cyber-Network-Packet-Sniffer
```


### 3. Install Dependencies

```bash
pip install -r requirements.txt
```


---

## ▶️ Run Project

Run:

```bash
python network_sniffer_gui.py
```


For Windows users:

Npcap must be installed for packet capturing.
